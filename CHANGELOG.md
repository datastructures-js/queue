## [3.1.0] - 2019-12-28
### Added
- `.clone()` to create a shallow copy of the queue.
- Enable creating the queue from an existing list
  - `new Queue(list)` enable passing a list in the constructor.
  - `Queue.fromArray(list)` static function.
